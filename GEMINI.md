# Antigravity Workspace Configuration (Talé)

- **Platform Context**: Hotel Booking web application
- **Theme**: Luxury coastal (neutral earthy tones like sand and terracotta, tactile textures like oak and stone, full-width panoramas, context-aware Dark Mode)
- **Frontend Stack**: Next.js, Tailwind CSS, Framer Motion, React Query, Redux
- **Backend Stack**: Node.js, Express.js, MongoDB (Mongoose), Passport.js
- **Payment Gateway**: PayMob
- **Media**: Cloudinary
