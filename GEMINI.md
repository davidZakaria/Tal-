# Antigravity Workspace Configuration (Talé)

- **Platform Context**: Hotel Booking web application
- **Theme**: True Soul of a Beach Resort - Modern luxury, elemental beauty, and high-end hospitality.
- **Visual Identity**: 
  - **Colors**: Soft nature-inspired palette. Crisp whites, `sand-light` (#fbf8f5 base), `sand` (#e3c8ab dunes), `terracotta` (#e2725b accent), `sapphire` (#06405a primary deep blue), and `turquoise` (#30c5d2 ocean highlight).
  - **Typography**: Elegant oversized Serif fonts (Playfair Display) for luxury headings. Highly legible, clean Sans-Serif (Inter) for body text and tactile UI components.
  - **Layout Constraints**: "Tactile haven", full-width panoramic layouts, off-grid aesthetic elements, highly confident use of negative space. Soft rounded corners for interactive components, and glassmorphic overlays over large media.
- **Frontend Core**: Next.js, Tailwind CSS (v4), Framer Motion for rhythmic fluid, wave-like ocean motion interactions. 
- **Backend Stack**: Node.js, Express.js, MongoDB (Mongoose), Passport.js
- **Payment Gateway**: PayMob
- **Media**: Cloudinary
